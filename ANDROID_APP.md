# Namma Kathey Android App

This workspace now contains a Kotlin Android project in `app/`.

## Open and run

1. Open this folder in Android Studio.
2. Let Gradle sync the project.
3. Select the `app` run configuration.
4. Run on an Android emulator or a physical Android device.

## What is implemented

- Kotlin + Jetpack Compose Android application.
- District selection flow.
- Hero gallery with bilingual Kannada/English content.
- Story reader with page navigation.
- Quiz flow with persisted badge unlocks.
- Badge profile screen.
- Statue/memorial finder screen prepared for future GPS integration.

The app is intentionally offline-first. It does not require API keys to run.
